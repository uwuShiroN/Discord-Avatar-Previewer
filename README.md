# Discord Avatar Preview

A simple Discord-style avatar preview tool made because I could never find a website that actually felt good to use.

Most avatar cropper websites were either:

* too bloated
* looked outdated
* didn't preview avatars inside a real Discord-like chat
* or just didn't let me adjust things the way I wanted

So I decided to make my own.

## Features

* Real-time Discord-style avatar preview
* Drag & resize circular crop area
* Multiple avatar size previews
* Chat scale simulation
* Dark / light mode
* English / 中文 / Mixed language UI
* Responsive resizable layout
* Instant PNG export

## Why I Made This

I wanted a tool where I could:

* quickly test how an avatar actually looks in Discord
* see readability at smaller sizes
* crop avatars more naturally
* and avoid using overly complicated editors

This project started as a small personal tool, but eventually turned into something polished enough to share.

## Built With

* HTML
* CSS
* Vanilla JavaScript

No frameworks. Just pure frontend.
